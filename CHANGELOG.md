# 更新日志

## [1.0.0] - 2025-12-18

### 新增
- 🎉 初始发布版本
- ✨ 支持多个Java版本切换（Java 7/8/11/17/25）
- 🖥️ 现代化WPF图形界面
- 🔍 自动检测当前Java版本
- 🚀 单文件发布，无需安装.NET运行时
- 🔐 自动请求管理员权限进行环境变量修改

### 技术细节
- 基于.NET 10.0构建
- 面向Windows x64平台
- 使用WPF框架提供现代化用户体验
- 支持环境变量修改以实现Java版本切换

### 使用方法
1. 下载JavaVersionSwitcher.exe
2. 以管理员身份运行程序
3. 点击对应的Java版本按钮进行切换
4. 重新启动命令行或IDE验证切换结果
